# Silent Mode Counter

A melancholic, Apple-style glassmorphism experience. Built with Next.js, Tailwind CSS, and Framer Motion.

## Features
- Hero section with parallax scroll and serif typography
- Glassmorphism counter button — "I Miss Her"
- Moments in Collision: CSS animation referencing elastic/inelastic collision physics
- Footer CTA: "Keep Moving Right"

## Stack
- **Next.js 14** (App Router)
- **Tailwind CSS**
- **Framer Motion**

## Deploy
Hosted on Vercel.
